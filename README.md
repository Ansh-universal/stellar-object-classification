# Stellar Object Classification

== Overview
   --------
This project classifies celestial objects which are :-
- Star
- Galaxy
- Quasar

using machine learning techniques on astronomical data.

== Technologies Used
   -----------------
- Python
- Pandas
- NumPy
- Scikit-Learn
- Random-Forest
- Streamlit

== Models Used
   -----------
- Logistic Regression
- SVM (Support Vector Machine)
- Knn (K-Nearest Neighbors)
- Naive - Bayes
- Decision Tree
- Random Forest
- XGBoost

== Deployment (Streamlit)
   ----------
Interactive Streamlit web application.
